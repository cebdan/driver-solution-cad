# Driver Solution CAD

CAD система с микроядром на основе Solution, поддерживающая историю построения, систему нод для скриптов, XTD GUI (на базе библиотеки xtd), OpenGL 2.1 для 3D рендеринга и терминал для взаимодействия с пользователем.

## Требования

- C++20 компилятор
- CMake 3.20 или выше
- Библиотека [xtd](https://github.com/gammasoft71/xtd) - современный C++20 фреймворк для GUI

## Установка xtd

```bash
# Клонировать репозиторий xtd
git clone https://github.com/gammasoft71/xtd.git
cd xtd

# Собрать и установить (следуйте инструкциям в репозитории xtd)
```

## Сборка проекта

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Запуск демо приложений

### Демо терминала
```bash
./demo_terminal
```

### Демо главного окна с меню
```bash
./demo_main_menu
```

Главное окно содержит меню:
- **File** → New 2D Document (Ctrl+N), Open, Save, Close, Exit
- **Edit** → Undo, Redo, Cut, Copy, Paste
- **View** → Zoom In/Out, Fit to Window
- **Tools** → Options, Terminal
- **Help** → About, Documentation

## Использование терминала

После запуска демо приложения откроется окно терминала. Доступные команды:

- `help` или `?` - показать справку
- `status` - показать статус решения
- `nodes` - список всех нод
- `history` - история построения
- `name` - имя решения
- `clear` - очистить терминал

Навигация:
- `Enter` - выполнить команду
- `↑` / `↓` - навигация по истории команд

## Основные возможности

- **Микроядро Solution** - базовый класс для всех решений CAD системы
- **История построения** - отслеживание операций с возможностью undo/redo
- **Система нод** - выполнение скриптов через ноды
- **XTD GUI** - графический интерфейс на базе библиотеки xtd
- **Терминал** - интерактивное окно для ввода команд и вывода информации
- **OpenGL 2.1** - рендеринг 3D объектов
- **Обмен данными** - Solution могут обмениваться данными, если реализованы соответствующие методы
